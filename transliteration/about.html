This is a transliteration tool which facilitates transliteration from Unicode Devanagari into various Roman transliteration schemes used for Sanskrit and vice versa. The current version provides conversion programmes for the following transliteration schemes: WX, SLP1, Velthuis, Harvard-Kyoto, Itrans and IAST (International Alphabet for Sanskrit transliteration).

The interface and the shell programs are written by Karnunakar.

Most of the transliteration programs are written by Amba Kulkarni.

All the programs are available under GPL. Contact ambapradeep@gmail.com for the source code.
